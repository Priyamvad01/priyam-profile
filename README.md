# Priyam Profile

Minimal cinematic AI and cloud engineer portfolio foundation built with React, Vite, Tailwind CSS, and Framer Motion.

## Phase 1 Scope

This setup is intentionally limited to the project foundation:

- Vite React application shell
- Tailwind CSS 4 with PostCSS
- Framer Motion animation presets
- Scalable source folders and import aliases
- Premium dark theme tokens
- Responsive container and spacing utilities
- Reusable base components for future sections
- First-person content direction for future portfolio copy

No full portfolio sections have been built yet.

## Scripts

```bash
npm install
npm run dev
npm run build
npm run lint
```
